
	The Page GUI will be a form:
		1. Course Name input of max 100 char
		2. Course Period input as numbers in hours 
			2.1 Can't take negative numbers
		3. Course Type Select of (lookup Course Types)
		4. Course Level Select of (lookup Course Levels)
		5. YouTube Playlist URL input of max 120 char
			5.1 Check if its valid Youtube Playlist URL
		6. Course Objectives input of 100 char with Plus Button to add more Objectives
		7. Course Content input of 100 Char with Plus Button to add more Contents
		8. Course References input of 3 inputs with Plus Button to add more References
			8.1 Reference name input of 100 char
			8.2 Reference Type Select (lookup Reference Types)
			8.3 Reference URL input of 150 char
		9. Course Pre-Requisite input of 2 inputs with plus Button to add more References
			9.1 pre-requisite name input of 100 char
			9.2 pre-requisite URL input of 150 char
		10. Description input of 200 char
		
		
	Requirments of the Page:
		1. Only Admins can access this page
